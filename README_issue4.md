# Запуск тестов

1. Откройте терминал и перейдите в директорию с файлом one_hot_encoder.py.
2. Запустите тесты с помощью команды:

```bash
python -m pytest -v result_ohe_pytest.py
```
Чтобы сохранить в файл txt, необходимо:
```bash
python -m pytest -v result_ohe_pytest.py > result_ohe_pytest.txt
```
